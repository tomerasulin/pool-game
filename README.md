# Classic 8-ball Pool Game

A pool game where you can play against a friend.

### To play
https://tomerasulin.github.io/pool-game/

The code is based on the following [video](https://www.youtube.com/watch?v=aXwCrtAo4Wc&ab_channel=freeCodeCamp.org), I added the following:
1. Handling the ball-in-hole functionality
2. Adding Player class to hold the players info such as: score, color and turns
